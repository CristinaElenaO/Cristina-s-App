# Cristina-s-App

A lightweight mobile-friendly reservation planner.

## Features
- Add reservations with person name, phone number, and date/time.
- View all upcoming reservations.
- Generate WhatsApp reminder links for:
  - 1 day before the reservation
  - The same day as the reservation

## How to use
1. Open `/home/runner/work/Cristina-s-App/Cristina-s-App/index.html` in a browser.
2. Add reservation details in the form.
3. Use the WhatsApp buttons to open pre-filled reminder messages.

Reservations are stored in browser local storage.
